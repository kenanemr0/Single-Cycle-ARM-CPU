Houses the EDA tool files necessary for synthesizing the design and deploying it to hardware. It contains the Intel Quartus II project files (.qpf, .qsf) and specific pin assignments required for the Intel DE1-SoC (Cyclone V) FPGA board. These files ensure the internal signals are correctly routed to the board's buttons, switches, and seven-segment displays.

Note That uploaded files utilizes the EDA tool files that are built for DE0 Board (Cyclone III), including its pin assignment.
